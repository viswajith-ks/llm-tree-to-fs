# LLM Tree to Filesystem

Converts ASCII directory tree diagrams generated by AI models (ChatGPT, Gemini, Claude) into actual file system structures.

## Web Interface

Use the browser-based version to generate a project structure and download it as a ZIP file.

## Python CLI

Run the script locally to generate files in the current directory.

**Usage:**

```bash
python python-cli/main.py [structure_file]
```

If no file is specified, it defaults to looking for `structure.md`.
