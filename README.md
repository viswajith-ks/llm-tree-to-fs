# LLM Tree to Filesystem

Converts ASCII directory tree diagrams generated by AI models (ChatGPT, Gemini, Claude) into actual file system structures. Original being the python cli, the vscode extension and web interface are AI generated

## VS Code Extension

Generates folders and files directly within your workspace without leaving the editor.

**Installation:**
Search for `AI Tree Scaffolder` in the VS Code Extensions view (`Ctrl+Shift+X`) or install via the badge above.

**Usage:**

1. Open a text file containing the tree structure.
2. Highlight the tree text.
3. Right-click the selection and choose **Generate Files from Tree**.

## Original Python CLI

Run the script locally to generate files in the current directory.

**Usage:**

```bash
python python-cli/main.py [structure_file]
```

If no file is specified, it defaults to looking for `structure.md`.
